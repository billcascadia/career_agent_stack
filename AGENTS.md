# Agent Registry

Use these roles as specialized operating modes. The orchestrator owns sequencing and user consent.

## 1. Orchestrator

File: `agents/orchestrator.md`

Owns onboarding, routing, progress, user approval boundaries, and handoffs between agents.

## 2. Privacy Guard

File: `agents/privacy-guard.md`

Checks whether outputs leak private source data, overclaim experience, or include unsafe assumptions.

## 3. Profile Builder

File: `agents/profile-builder.md`

Turns setup answers, resumes, portfolio links, and work history into structured source truth.

## 4. Opportunity Scout

File: `agents/opportunity-scout.md`

Finds or ingests opportunities, filters for role and location fit, and sends promising roles to evaluation.

## 5. JD Analyst

File: `agents/jd-analyst.md`

Extracts requirements, risks, hidden filters, ATS terms, and hiring signals from a job description.

## 6. Resume Tailor

File: `agents/resume-tailor.md`

Produces tailored resumes from source truth and the neutral resume template.

## 7. Application Assistant

File: `agents/application-assistant.md`

Drafts application form answers, cover letters, salary answers, and upload checklists.

## 8. Outreach And Follow-Up

File: `agents/outreach-followup.md`

Drafts warm path messages, recruiter notes, follow-ups, and relationship tracking.
