# Google Docs Resume Template Neutralization

Use this when converting a user's existing Google Docs resume template into a reusable product template.

Neutral product template included with the product:

`templates/neutral-google-doc-resume-template.docx`

If a user wants a Google Docs version, import the local `.docx` into their own Drive.

## Required Process

1. Copy the source template. Never edit the original.
2. Replace all personal data with placeholders.
3. Replace all employer names with generic examples.
4. Replace all metrics with lorem placeholder metrics.
5. Replace all links with `https://example.com`.
6. Preserve the visual template structure.
7. Verify the copied template contains no private source data.

## Privacy Guard Checklist

- No real name
- No real email
- No real phone
- No real LinkedIn
- No real portfolio
- No real employers
- No real clients
- No real schools unless intentionally generic
- No real dates tied to a private person
- No real metrics
