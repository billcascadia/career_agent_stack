1. What This Stack Does

The stack is a career production line. It helps a user search for jobs, evaluate fit, tailor materials, track applications, create warm-path outreach, and prepare follow-ups. It does not submit applications or send messages without approval.

  • Profile Builder turns source materials into verified career truth.
  • Target Strategy defines the market, role lanes, filters, and deal breakers.
  • Opportunity Scout finds and filters roles.
  • JD Analyst extracts requirements, keywords, and risks.
  • Match Scorer estimates ATS, cold interview, and warm-path probabilities.
  • Resume Tailor creates role-specific drafts from the included template.
  • Application Assistant prepares answer drafts and application packets.
  • Warm Path Builder drafts referral and recruiter messages.
  • Tracker And Follow-Up keeps next actions visible.

2. Agent Production Line

Each stage has a bounded job. The orchestrator moves work from one stage to the next so the user can see what is happening and approve the high-impact decisions.

1. Setup OrchestratorExplains the stack, creates config files, and sets approval boundaries.
2. Profile BuilderBuilds a source-of-truth profile from verified resume and portfolio facts.
3. Target StrategyDefines titles, industries, compensation, location, and career-change lanes.
4. Opportunity ScoutFinds, ingests, filters, and deduplicates job opportunities.
5. JD AnalystExtracts requirements, seniority signals, keywords, and posting risks.
6. Match ScorerRanks roles by ATS Match %, Cold Interview %, Warm Path %, and Overall Fit.
7. Resume TailorCreates truthful role-specific resumes using the local neutral template.
8. Application AssistantDrafts form answers, cover notes, and packet checklists for review.
9. Warm Path BuilderFinds referral angles and drafts user-approved outreach.
10. Tracker And Follow-UpUpdates statuses, follow-up dates, outcomes, and next actions.
11. Interview PrepTurns each target role into talking points, story banks, and questions.
12. Learning LoopUses outcomes to improve the next search, scoring, and resume pass.

3. What The Orchestrator Should Say

The setup experience should feel deliberate. The agent should explain the system before asking for personal information.

Welcome. This stack is a career production line, not a one-off resume generator. We will set up your source truth first, then define your target market, then use agents to search, score, tailor, track, and follow up. The agent can search for roles, estimate ATS and interview probability, draft resumes, prepare application answers, and organize follow-ups. The agent will not submit applications, send messages, invent experience, or overwrite your source truth without your approval. You can use local files, Google Drive, or both. The default resume template is included in this folder, so you do not need access to anyone else's Drive. Setup can be completed in stages. Blank answers are allowed. Anything we create can be edited later.
